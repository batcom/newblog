## 常用正则

1. 金额正则

```js
// 不允许零值 如0,0.0,0.00等 两位小数点
const MONEY = '/^(([1-9]\d*)(\.\d{1,2})?)$|(0\.0?([1-9]\d?))$/';
// 允许零值
const MONEY_WITH_ZERO = '/^(([1-9]\d*)|0)(\.\d{1,2})?$/';

```

测试，浏览器F12打开直接

```js
['-1', '0','0.00', '0.01', '0.10', '0.101', '1', '01', '10', '10.', '10.1', '10.10'].map(str => {
    return /^(([1-9]\d*)|0)(\.\d{1,2})?$/.test(str)
})
```
